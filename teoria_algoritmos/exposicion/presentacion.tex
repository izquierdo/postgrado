  \frametitle{Ejemplos}

  Problema de satisfactibilidad: ¿es posible satisfacer las siguientes expresiones?

  \begin{itemize}
  \item $v_1 \vee v_2$
  \item $(v_1 \vee \neg v_2) \wedge (v_2 \vee v_3)$
  \item $(v_1 \wedge \neg v_2) \wedge (v_2 \wedge v_3)$
  \end{itemize}
}

  \frametitle{Un ejemplo más complicado}

  \end{itemize}
